## PYTHON FOUNDATION

#Level 1: Factorial Calculator
Description:  This program calculates the factorial of a number using recursion.
It takes user input and computes the factorial based on the mathematical definition:
                0! = 1
                n! = n × (n-1)!

#Level 2: Find Maximum Number in a List
Description: This program finds the largest number in a predefined list without using built-in functions.

Approach:
         ~ Assume the first element is the maximum
         ~ Loop through the list
         ~ Update the maximum whenever a larger value is found



#Level 3: CLI Tic-Tac-Toe Game
Description: This is a command-line (CLI) based two-player Tic-Tac-Toe game.

Features
         - Interactive board display
         - Player turn switching (X and O)
         - Input validation (invalid input, occupied positions)
         - Win detection (rows, columns, diagonals)
         - Draw condition handling
